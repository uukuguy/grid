## Accomplishments
- `L4_ALLOW_TRIM_P4` env var controls `allow_trim_p4` budget flag (defaults to `false` for backward-compat)

## Decisions Made
- **Backward-compat dep format:** MCP resolution in Step 6 handles both flat string and dict deps since it reads raw `skill_instructions` before normalization
- **Burst detection is passive:** Only logs WARNING — no automatic throttling or queue resize. Operators react to warnings.
- **Reference-mode flag:** `reference_mode: true` in event payload lets consumers detect new format; `payload_keys` list helps debug without full payload
