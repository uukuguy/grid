## Sampling Rate

- **After every task commit:** Run `cargo check -p <touched-crate>` (≤15 s)
- **After every plan wave:** Run targeted contract test subset (e.g., `pytest tests/contract/cases/test_chunk_type_contract.py -v --runtime=grid` after Wave 2)
- **Before `$gsd-verify-work`:** Full suite must be green AND `make verify-runtime` PASS for 主力档+样板档 4 runtimes
- **Max feedback latency:** 15 s (quick) / 120 s (full)
