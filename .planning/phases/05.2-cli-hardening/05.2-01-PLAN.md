### H2-3: Session Lifecycle Polish (CLI-05)

#### T-01.16: Add `session kill --purge` flag
- **File:** `crates/grid-cli/src/commands/session.rs`
- **Action:**
  - Add `--purge` flag to `session kill` subcommand
  - Without `--purge`: soft-delete (mark session state as deleted, keep proto sync markers)
