**D59 (L2-11) — Makefile mcp-orch-start port env var:**

In `Makefile` L859–861, change the `mcp-orch-start` target to read port from env var:

Current (L859–861):
```makefile
mcp-orch-start:
	cargo run -p eaasp-mcp-orchestrator -- --config tools/eaasp-mcp-orchestrator/config/mcp-servers.yaml --port 18082
```

Replace with:
```makefile
L2_MCP_ORCH_PORT ?= 18082

mcp-orch-start:
	cargo run -p eaasp-mcp-orchestrator -- --config tools/eaasp-mcp-orchestrator/config/mcp-servers.yaml --port $(L2_MCP_ORCH_PORT)
```

The `?=` assignment means: use env var `L2_MCP_ORCH_PORT` if set; otherwise default to 18082. This matches the L2-08 `?=` pattern used elsewhere in the Makefile (e.g., `VERIFY_ENDPOINT ?= http://localhost:50051` on L872).

The env var `L2_MCP_ORCH_PORT` can be set in `.env` or shell:
```bash
export L2_MCP_ORCH_PORT=19082
make mcp-orch-start  # uses 19082
```
