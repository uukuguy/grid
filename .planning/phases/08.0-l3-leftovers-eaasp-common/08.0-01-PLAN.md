```python
def _sanitize_errors(errors: list[dict[str, Any]]) -> list[dict[str, Any]]:
    """Strip non-JSON-serializable objects from Pydantic error dicts."""
    clean: list[dict[str, Any]] = []
    for err in errors:
        safe: dict[str, Any] = {}
        for key, value in err.items():
            if key == "ctx" and isinstance(value, dict):
                safe[key] = {
                    ctx_key: (
                        str(ctx_val) if isinstance(ctx_val, BaseException) else ctx_val
                    )
                    for ctx_key, ctx_val in value.items()
                }
            elif isinstance(value, BaseException):
                safe[key] = str(value)
            else:
                safe[key] = value
        clean.append(safe)
    return clean
```

| L3 → eaasp_common package | Local package dependency via pip install -e; no network boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08.0-01 | Denial of Service | `sanitize_errors()` in `errors.py` | accept | Function processes Pydantic error dicts (small payloads, no external network input). Recursion depth bounded by dict nesting. No amplification vector. |
| T-08.0-02 | Information Disclosure | `sanitize_errors()` in `errors.py` | mitigate | Function already converts `BaseException` objects to strings, preventing traceback leakage. The import path `eaasp_common.errors.sanitize_errors` does not expose internal L3 state. |
| T-08.0-03 | Elevation of Privilege | L3→eaasp_common import | accept | Local pip install -e dependency; same trust domain. No auth/perm escalation path. |
| T-08.0-04 | Spoofing | eaasp-common package identity | accept | Package is local (`pip install -e` from directory); no PyPI distribution. No supply-chain injection risk in this phase (L2-12 will re-evaluate). |
