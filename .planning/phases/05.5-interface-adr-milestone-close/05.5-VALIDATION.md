## Sampling Rate

- **After every task commit:** Run the task's specific `<verify><automated>` block (static grep + targeted test combo, < 10s).
- **After every plan wave:** Run `python3 .adr-plugin/scripts/adr_lint.py` (full lint) + `cargo test -p grid-cli` (~30s combined).
- **Before `$gsd-verify-work`:** Phase gate must be green:
  ```bash
  python3 .adr-plugin/scripts/adr_lint.py            # 0 FAIL, F3 WARN ≤ 13
  cargo test -p grid-cli                              # 147+ tests PASS
  cargo build -p grid-cli --features studio           # studio feature builds clean
  cargo check --workspace                             # workspace check passes
  cd tools/eaasp-l2-memory-engine && uv run pytest tests/test_vector_index.py
  ```
- **Max feedback latency:** 30 seconds.
