## Other decisions (agent's discretion, not asked)

- **Buffer cap**: 1000 entries/server (D-11). `tail -1000` industry default.
- **broadcast::Sender capacity**: 256 (within 100-1024 agent-discretion range).
- **LogEntry level inference**: case-insensitive prefix match for `error/err/warn`.
- **Output format inheritance**: text default, JSON when `--output json` OR non-TTY stdout (per Phase 3.7.1 D-07).
